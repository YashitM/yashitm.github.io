Some of my Projects

Speech Recognition using Arduino

Equipment used: Arduino UNO R3, HC-05 Bluetooth Module, an Android Phone with AMR-Voice App installed, LCD Display, Jumper Cables etc.

Language(s) used: Embedded C

About: This device displays the answer to various mathematical expressions the user says. The user gives the voice input to the Android App (AMR-Voice) and then this app forwards the data to the Bluetooth Module attached to the arduino. The Arduino then processes the data and then displays the final output onto the LCD Display attached to it. As of now, it does basic mathematical calculations, let's us know the surrounding temperature and also acts as a switch for various LEDs.

Useful Links: Github Repository

The Retro Snake Game

Language(s) used: C++

About: This is a very basic B&W Snake Game. The snake can be moved using the W, A, S and D keys. It's size increases whenever the head comes in contact with a fruit that is randomly placed in the frame. It has 3 levels of difficulty (that vary according to the speed of movement of the snake).

Pollution Sensing using Arduino

Library Management System

Languages used: C++

About: A handy and interactive Library Management System with features of storing, adding and deleting various users. Along with users, new books can be added, lost books can be deleted and books can be borrowed by any of the users (depending on the availability of books). The borrowed book, if not returned within a specified period of time, will result in a fine (depending on the number of days by which the user is late to return the book by) that the user will have to pay.
